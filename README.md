## Questlog T1
Убирает рыжие, T3, T2 шмотки и фиксит статы в синергиях рун.
## Установка
1. Установи Tampermonkey [Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?pli=1) [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
   - Убедись, что включен Developer Mode + Enable user scripts: [Гайд как включить](https://www.tampermonkey.net/faq.php?q=Q209)
3. Создай пустой скрипт.<img width="298" height="320" alt="image" src="https://github.com/user-attachments/assets/07bc5363-f34a-4a0a-917e-0cf2c72434f4" />
4. Вставь содержимое [script.js](script.js) в созданный скрипт.
5. Сохрани File -> Save<img width="217" height="426" alt="image" src="https://github.com/user-attachments/assets/3725aaa2-59be-428e-b86e-f860f72914b1" />
6. Заходи в квестлог.
